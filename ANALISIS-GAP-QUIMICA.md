**FALTA 3 EJEMPLOS EXPERIMENTALES COMPLETOS:**

1. **Titulación ácido-base:**
   - Problema: Determinar concentración HCl desconocida
   - Materiales: Bureta, NaOH 0.1M, fenolftaleína
   - Procedimiento: Titular 25mL HCl con NaOH hasta viraje
   - Datos: Gasto 30mL NaOH
   - Cálculo: M₁V₁ = M₂V₂ → [HCl] = 0.12 M
   - Variables: Dependiente (volumen NaOH), independiente (concentración HCl)

2. **Equilibrio químico (Le Chatelier):**
   - Reacción: CoCl₂(azul) + 6H₂O ⇌ [Co(H₂O)₆]²⁺(rosa) + calor
   - Experimentos: Agregar HCl, enfriar, calentar
   - Observaciones cambio color → predicciones confirmadas

3. **Cinética - Velocidad vs Temperatura:**
   - Reacción: Na₂S₂O₃ + HCl (formación S turbio)
   - Variable independiente: Temperatura (10°C, 20°C, 30°C, 40°C)
   - Variable dependiente: Tiempo hasta turbidez
   - Gráfica: T↑ → tiempo↓ (velocidad↑)
   - Hipótesis: Arrhenius predice relación exponencial
