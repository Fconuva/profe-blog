# Guía de imágenes para Maus — Prueba NM4

## Instrucciones generales
Las viñetas deben escanearse o fotografiarse del libro **Maus: A Survivor's Tale** (Volumes I y II, Art Spiegelman).

- **Formato:** JPG, resolución mínima 800px de ancho
- **Carpeta destino:** `lecturas/contenidos/imagenes/maus/`
- **Nombrado exacto:** usar los nombres de archivo indicados abajo

---

## Notas legales
Uso amparado por la Ley 17.336, artículo 71 B (fines educativos, sin ánimo de lucro, fragmentos breves).

## Alternativa si no se dispone de imágenes
La prueba funciona sin imágenes: las descripciones textuales de las viñetas (en el JSON) son suficientemente detalladas para responder todas las preguntas. Las imágenes enriquecen la experiencia pero no son bloqueantes.
