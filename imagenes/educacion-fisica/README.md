## Fuentes recomendadas
- Unsplash/Pexels para fotografías (licencia libre)
- Storyset/Undraw para ilustraciones SVG
- Flaticon para iconos (atribución)

## Especificaciones técnicas
- Formato: PNG para ilustraciones, JPG para fotos
- Tamaño máximo: 300 KB por imagen
- Dimensiones: 1200-1600px ancho máximo
- Usar `loading="lazy"` en HTML
- Alt text descriptivo obligatorio
